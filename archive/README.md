# Historical Archive Policy

The original Stage71 full integrated repository is intentionally not expanded into this canonical GitHub repo by default. It should be attached as a release asset if full forensic reconstruction is required.

Stage-level knowledge is preserved in:

- `docs/stages/`
- `docs/concepts/`
- `manifests/stage_lineage_manifest.json`
- `manifests/file_inventory.json`
- `manifests/archive_manifest.json`
