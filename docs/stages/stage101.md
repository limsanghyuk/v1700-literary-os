# Stage101 - Cross-Lineage Absorption & Scenario Room Integration

Stage101 is the first post-RC evolution stage after Stage100 FINAL FIXED.

It does not merge V430 wholesale. It absorbs V430 scenario-room strengths through V1700 contracts, fixtures, branchpoint trace, release evidence, and release gates.

## Scope

- Stage101.0: Cross-Lineage Preflight & Source Availability Lock
- Stage101.1: Scenario Room Contract Layer
- Stage101.2: Beat / Action / Dialogue / Prop Cue Integration
- Stage101.3: Dual-Mode Regression + Absorption Release Gate

## Absorbed Concepts

- Scenario Room Planner
- Scene Beat Board
- Investigation / action-driven movement
- Dialogue / silence cue
- Prop-led reveal cue

## Non-Goals

- No V430 direct runtime merge.
- No untraced V430 imports.
- No live provider calls in release gates.
- No Node2 raw reveal access.
- No raw manuscript provider export.
- No product API expansion before Stage103.

## Release Blockers

- Stage100 baseline fails.
- V430 untraced merge is detected.
- Scenario room bypasses V1700 contracts.
- Prose/scenario metrics are conflated.
- Prop reveal bypasses reveal budget.
- Dialogue cue accesses raw reveal.
- Provider live call count in release gate is nonzero.
- Missing manifest, test, release evidence, or branchpoint trace.
