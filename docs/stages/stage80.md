# Stage80 — Korean Drama Story & Macro Plot Composition Engine

Stage80 corrects a critical hierarchy error: the whole story, macro plot, broadcast episode, micro plot, sequence, and scene are not the same concept.

## Core hierarchy

```text
Series Story
→ Macro Plot Architecture
→ Broadcast Episode Composition Map
→ Per-Episode Micro Plot Set
→ Sequence Chain
→ Scene Chain
→ Prose / Script Rendering
```

## Definitions

- **Series Story**: the total transformation from the beginning to the end of the work.
- **Macro Plot**: a large phase or world/institutional arc inside the whole story, such as apprenticeship, exile/recomposition, and return with a new function.
- **Broadcast Episode**: a televised/chapter unit that advances the whole story inside one macro plot.
- **Micro Plot**: a small story line inside one broadcast episode.
- **Sequence**: a meaningful chain of scenes that executes a micro plot.
- **Scene**: the smallest dramatic unit.

## Acceptance

Stage80 passes only if the engine keeps these layers separate and produces supporting-character web, micro plots, sequence chains, and scene chains.
