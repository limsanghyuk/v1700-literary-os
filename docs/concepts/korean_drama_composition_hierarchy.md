# Korean Drama Composition Hierarchy

This document records a core V1700 literary logic: Korean drama composition is not a flat episode list.

```text
Series Story != Macro Plot != Broadcast Episode != Micro Plot != Sequence != Scene
```

## Series Story

The total change of the work: what changes between the beginning and the end for the protagonist, world, relationship order, and event order.

## Macro Plot

A major phase of the story. For example, a heroine may pass through an apprenticeship world, an exile/recomposition world, and a return-with-new-function world. These are not episodes; they are large plot blocks.

## Broadcast Episode

A broadcast/chapter unit. Each episode advances a specific portion of the series story inside a macro plot.

## Micro Plot

A small plot line inside a broadcast episode. It may feel like a small episode, but it is not the whole-story episode.

## Sequence

A meaningful unit that organizes scenes around one micro plot function.

## Scene

A local dramatic unit inside a sequence.
