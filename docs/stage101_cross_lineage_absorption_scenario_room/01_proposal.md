# Stage101 Proposal

## Title

V1700 Stage101 - Cross-Lineage Absorption & Scenario Room Integration

## Purpose

Stage101 promotes the Stage100 RC into a safer cross-lineage evolution path. It evaluates V430-style scenario-room strengths, blocks untraced runtime merges, and absorbs only contract-safe concepts into the V1700 Branchpoint OS.

## Three-Expert Council

### Principal Architect

Stage101 must not become a broad V430 merge. V1700 already owns the literary operating-system authority: Narrative Physics, Branchpoint Governance, Node2 surface boundary, provider-zero release gates, and dual-mode evaluation. V430-like strengths are useful only when recompiled as V1700 contracts.

### Principal Compiler

Stage101 must compile into small, testable modules:

- `cross_lineage`
- `scenario_room`
- `stage101`
- `stage101_release_gate`
- Stage101 manifests and release evidence

Every concept must have a data contract, fixture proof, test, release evidence, and branchpoint trace.

### Principal System Engineer

Stage101 must preserve the mandatory pre-development invariants:

- provider default calls remain zero
- live provider calls in release gate remain zero
- Node2 raw reveal access remains zero
- raw manuscript provider leakage remains zero
- branchpoint lineage remains preserved
- GitNexus remains optional and Python fallback remains required

## Proposed Absorption Candidates

- Scenario Room Planner
- Scene Beat Board
- Investigation / action-driven movement
- Dialogue / silence cue engine
- Prop-led reveal cue system

## Explicit Non-Goals

- No V430 direct runtime import.
- No V430 code merge without branchpoint trace.
- No product API expansion in Stage101.
- No provider live call expansion.
- No rewrite of Stage100 RC authority.

## Success Criteria

Stage101 succeeds only when:

- Stage100 baseline gate passes.
- V430 source status is recorded.
- V430 untraced merge is false.
- Scenario-room contracts pass.
- Dual-mode prose/scenario separation remains intact.
- Stage101 release gate passes.
- Main release gate passes.
- Repo doctor passes.
- Clean ZIP packaging passes.
